# Project description
```bash
The project consists of five games:
I. Game: "Parity check"
II. Game: "Calculator"
III. Game "GCD"
IV. Game "Arithmetic progression"
V. Game "Is it a prime number?"
```
# Setup
```bash
$git clone https://github.com/Barllou/frontend-project-44.git
$cd frontend-project-44
$make install
```
# Launch games
```bash
make 'name game' or...
brain-'name game'

gameList:
   - even
   - calc
   - gcd
   - progression (for make 'progr')
   - prime
```

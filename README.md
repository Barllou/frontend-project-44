# Project description
```bash
The project consists of five games:
I. Game: "Parity check"
II. Game: "Calculator"
III. Game "GCD"
IV. Game "Arithmetic progression"
V. Game "Is it a prime number?"
```
# Setup
```bash
Make install
```
# Launch games
```bash
make _name game_ or...
brain-_name game_

__gameList = {
    name: even,
    name: calc,
    name: gcd,
    name: progression, (for make -progr_)
    name: prime
};__
```
